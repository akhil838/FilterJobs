# Filter Jobs

Filter Jobs is a browser extension for Safari that simplifies your job search by automatically hiding fake, spammy, and overly promoted job postings on popular platforms like LinkedIn, Indeed, and Glassdoor. With just one click, you can eliminate the clutter and focus on genuine opportunities that matter.

# How to Install
1. Download the dmg from the [releases](https://github.com/akhil838/FilterJobs/releases)
2. Open the dmg, Drag and Drop the FilterJobs app into Applications folder
3. Allow installation from external sources in settings
4. Open the app
